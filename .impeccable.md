# Impeccable — Claudinio Code

## Design Context

### Users
Desenvolvedores profissionais (o próprio Victor como primeiro usuário) usando um app desktop macOS de agente de código (Tauri 2 + SolidJS). Contexto: sessões longas de trabalho, lado a lado com editor/terminal. O trabalho a fazer: delegar tarefas de código a um agente, acompanhar o raciocínio/ferramentas em tempo real e aprovar diffs com confiança.

### Brand Personality
Calmo, preciso, artesanal. Alinhado ao claudin.io / estética Anthropic: quente sem ser fofo, técnico sem ser frio. Deve evocar confiança e foco — nunca ansiedade ou ruído. Anti-referências: VS Code genérico, dashboards SaaS com neon, "AI slop" roxo-ciano.

### Aesthetic Direction
- **Tema**: dark + light seguindo o sistema. Dark = carvão quente (#141210 base); light = papel quente (#faf8f5).
- **Paleta**: neutros quentes (tint terroso), accent terracota #d97757 usado cirurgicamente (ações primárias + estado do agente). Nada de gradientes decorativos, glassmorphism ou glow.
- **Tipografia**: Inter (UI, decisão aprovada pelo usuário) + JetBrains Mono (código, tools, paths). Escala densa 13/12/11px, micro-labels uppercase tracking-wider.
- **Referências**: Linear, Warp, apps da Anthropic — densidade calma, hairlines, hierarquia por tipografia e espaço, não por caixas.
- **Chat**: documento contínuo sem bolhas; timeline do agente em linhas compactas colapsáveis.

### Design Principles
1. **Silêncio visual**: o conteúdo do agente é o protagonista; chrome quase invisível (hairlines, superfícies próximas).
2. **Accent raro**: terracota só onde há decisão ou vida (enviar, aprovar, agente trabalhando).
3. **Densidade calma**: informação densa mas escaneável — 1 linha por step, expandir sob demanda.
4. **Motion funcional**: CSS-only, 120–160ms, apenas para mudanças de estado; respeitar prefers-reduced-motion.
5. **Idioma**: toda a copy da UI em português.
