| Property/type mismatches | 13 | Fix per file: `ChatPanel.test.ts` — `TimelineNode.text` → add `text` property to type or use correct shape. `DiffViewer.tsx` — `containerRef` possibly undefined → non-null assertion. `DiffViewer.test.tsx` — `mockImplementationOnce` → use `mockImplementation`. `FileTree.test.tsx` — cast mock type properly. `FileEditorModal.tsx` — Monaco v0.55.1 deprecated `languages.typescript` namespace → use alternative API. `theme.ts` — return type mismatch. `theme.test.ts` — expression not callable (never type). |
